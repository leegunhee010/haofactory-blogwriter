# 윈차이나 사실 기준서 (중국 비자 · 중국 제출용 아포스티유)

윈차이나는 '중국 전문' 채널이다. ① 중국 비자(취업·유학·상용·가족방문 등)와 ② 중국 제출용 아포스티유/번역공증, 이 두 축으로만 글을 쓴다.
가격·필요서류·소요기간은 아래 표에서만 인용하고, 표에 없으면 지어내지 말고 '서류·일정 확인 후 안내'로 열어 둔다.

## 핵심 규칙
- 중국은 2023년 11월 헤이그 협약 가입 → 영사관 영사인증 폐지, 아포스티유로 대체. 순서: 발급 → 번역 → 공증 → 아포스티유.
- '중국 영사인증' 문의가 와도 현재는 아포스티유로 진행한다고 바로잡아 안내.
- 제출처가 실물 스티커 아포스티유를 요구하는 경우가 많다 → 전자 아포보다 실물 발급 선호(특히 취업·유학·거류증·법인).
- 여권 영문명 100% 일치(이름·생년월일은 여권 기준). 접수 시 여권 사본 필수.
- 컬러 출력 필수(학력·범죄경력 등). 범죄수사경력회보서는 원본 컬러. 학교 홈페이지 전자증명서 PDF 불가.
- 중국 학력서류: 학신망(chsi.com.cn)에서 졸업/학위번호 조회 가능한 사람만 현지 공증·아포스티유 진행.
- 범죄경력: 이력 없으면 범죄·수사 통합 발급, 수사이력 있으면 범죄만 분리 발급 권장.
- 가족·혼인관계증명서: 상세본 + 주민등록번호 공개 발급 필수.
- 단정 금지: "무조건 N일·100% 발급" 대신 "평균 약 N영업일 / 영사 심사에 따라 변동".
- 취업·장기유학 등은 본인 방문 지문등록·최소 2주 전 신청 권장을 반드시 안내.

## 중국 비자 단가 (일반 / 긴급 / 특급, 원)
- 상용 단수(30/60/90일, 3개월 1회): 130,000 / 180,000 / 210,000
- 상용 더블(6개월 2회): 160,000 / 210,000 / 240,000
- 상용 복수(1/2/3년 다회): 200,000 / 250,000 / 280,000
- 취업(Z, 3개월 1회): 130,000 / 180,000 / 210,000
- 유학(X1, 3개월 1회): 130,000 / 180,000 / 210,000
- 유학(X2, 입학허가 일수 1회): 130,000 / 180,000 / 210,000
- 가족방문(S1): 160,000 / 200,000 / 230,000
- 친인척방문(Q1 / Q2): 160,000 / 200,000 / 230,000
- 친인척 Q2 복수(2~3년 다회): 230,000 / 280,000 / 320,000
- 미국인(60일 / 10년): 360,000 / 410,000 / 460,000
- 소요기간: 서류 수취 익일 기준 평균 약 8일(영업일 5~9일). 비자센터 예심 강화로 +2~3일. 최소 2주 전 신청 권장.
- 지문등록 필요: Z·X1·S1·Q1(최근 5년 내 동일 여권 서울센터 등록 이력 있으면 면제). 일반 단수·더블·상용·복수는 면제(비대면 접수).

## 비자 종류 핵심
- 유학: X1=180일 초과 장기 / X2=180일 이하 단기(단수). 필수: 입학허가서·JW202(또는 JW201)·(필요 시)DQ.
- 취업: 중국 현지 취업허가서(Work Permit Notification) 선행 → Z비자 → 입국 후 거류증 전환. 취업허가 신청 단계에서 학력·경력서류 아포스티유가 자주 필요. 학력·경력서류 오류 시 취업허가 심사 자체가 중단될 수 있음(정보 일관성이 핵심).
- 상용복수 조건: 1년(중국비자 발급 1회+), 2년(1년복수 1회+ 또는 비자 3회+), 3년(1년복수 2회/2년복수 1회/방문 5회 중 하나). 초청장에 방문목적·반복출장 사유 구체 기재. 만 60세 초과 심사 강화.
- 무비자: 30일 이내 단기 출장·미팅·시장조사·전시회 참관은 무비자 입국 가능. 비자런 반복은 입국심사에 불리.
- 가족동반(S/Q): 가족관계증명서 중문 번역공증 + 아포스티유까지 요구되는 경우가 많음.

## 중국 제출용 아포스티유 단가 (번역+공증인증+아포스티유 포함가, 배송료 별도, 원)
- 취업비자용: 범죄경력 아포 88,000 / 졸업증명서 아포 88,000 / 경력증명서 아포 88,000 / 패키지 범죄+졸업 176,000 / 범죄+졸업+경력 264,000
- 거류허가용: 가족관계 아포 88,000(상세+주민번호 공개) / 범죄경력 아포 88,000 / 패키지 (가족 또는 혼인)+범죄 176,000
- 결혼용: 혼인관계 아포 88,000 / 가족관계 아포 88,000 / 여권 아포 88,000(컬러 스캔·서명 필수) / 패키지 혼인+여권 176,000
- 법인 서류: 법인등기(등기사항전부증명서) 아포 77,000(번역료 별도) / 사업자등록증 아포 88,000 / 위임장 아포 110,000(사실공증)
- 중국 현지 학교서류(중국 발급): 성적/졸업/학위증명서 각 198,000원 · 약 2~3주(현지 번역 → 공증 → 외사판공실 아포스티유)
- 아포 진행 절차: 발급(정부24/주민센터/홈페이지 발급본, 컬러 출력) → 번역 → 공증(번역공증/사실공증) → 아포스티유(외교부).

## 발행 전 점검
중국 인증=아포스티유로 안내(영사인증 표현 잔존 X) / 비자·아포 가격이 위 표와 일치 / 소요기간·배송료 별도 명시 / 지문등록 대상(Z·X1·S1·Q1)·신청 시점 안내 / 발급 형태(컬러·전자증명서 불가·상세+주민번호·학신망) 정확 / '가견적·발급 보장' 금지.
표·규칙에 없는 신규 케이스는 자동 생성하지 말고 '일정·서류 확인 후 안내'로 둔다.
